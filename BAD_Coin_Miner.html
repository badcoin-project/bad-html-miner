BadCoin HTML Miner

Mine BAD from your browser. Yescrypt only. Pool mining. 100% client-side.
Performance expectations. Browser yescrypt runs at ~500–700 H/s per thread on Apple Silicon, ~50–200 H/s on older / non-ARM hardware. With multiple workers, total throughput is a few KH/s per machine. That is not competitive with native CPU miners, let alone the wider network. This is a community / learning / onboarding device, not a profit center. Expected earnings are small. The point is the experience: "I mined BAD from a browser tab."
Why no solo mode? Solo mining at browser hashrate against the BadCoin network's full difficulty means expected time to find a block is years to never. For real solo mining, use the BadCoin Core wallet's Mining tab: native code, full CPU hashrate, connects directly to your own badcoind. This browser tool is pool-only by design.
